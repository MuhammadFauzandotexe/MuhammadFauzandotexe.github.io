Tentang Diri

Sebagai seorang yang berminat dalam pengembangan perangkat lunak, saya memiliki pengalaman dan keahlian dalam bahasa pemrograman Java dan pengembangan backend. Saya tertarik dengan bagaimana sebuah masalah pengguna dapat diselesaikan melalui code program. Saya senang dan selalu siap untuk belajar hal-hal baru yang dapat meningkatkan kemampuan dan kinerja saya di bidang pengembangan software sesuai dengan tren teknologi. Saya percaya dengan kemampuan yang saya miliki, saya dapat memberikan kontribusi positif dalam mengembangkan solusi software yang inovatif dan berkualitas tinggi.

Saya memiliki pengalaman dalam menguasai beberapa framework seperti Spring Boot, Spring Security, Spring AOP, Spring MVC, Quarkus, serta penggunaan JPA dan Hibernate ORM. Selain itu, saya juga memiliki pengalaman dalam pengelolaan database seperti MySQL dan PostgreSQL, serta Redis. Kemampuan saya juga meliputi merancang REST API dan Micro Architecture.Selain itu, saya juga terampil dalam menggunakan version control seperti Git, Github, dan Gitlab. Saya memiliki beberapa aplikasi portofolio yang saya bagikan melalui akun Github, dan saya konsisten dalam memperbarui portofolio saya dengan proyek terbaru dan terbaik.
